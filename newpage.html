Important FAQ

What is the Document Object Model(DOM)?

The DOM (Document Object Model) is a programming interface for web documents. It represents the structure of a document as a tree of objects, where each node corresponds to an element, attribute, or text within an HTML or XML document. This tree-like structure allows each part of the document to be accessed and manipulated programmatically.

How do you select an element from the DOM?

document.getElementById() is used to select an element by its unique id attribute. For example, document.getElementById('myId') selects the element with the id of "myId". document.querySelector() allows you to select the first element that matches a CSS selector. For instance, document.querySelector('.myClass') will select the first element with the class "myClass", while document.querySelector('#myId') will select the element with the id "myId". document.querySelectorAll() selects all elements that match a CSS selector and returns a NodeList, which can be iterated over. For example, document.querySelectorAll('.myClass') selects all elements with the class "myClass".

What is event delegation in the context og the DOM, and whuy it is useful?

Event delegation in the context of the DOM refers to a technique where a single event listener is added to a parent element rather than individual child elements. When an event (like a click) occurs on one of the child elements, the event "bubbles up" (propagates) to the parent, where it is handled. This approach leverages event bubbling in the DOM.

How do you manipulatale an elements attributes and styles using the DOM?

To manipulate an element's attributes and styles using the DOM in JavaScript, you can utilize several methods and properties. For attributes, the getAttribute() method allows you to retrieve the value of a specific attribute, while the setAttribute() method is used to change or add an attribute to the element. If you need to remove an attribute, the removeAttribute() method will effectively delete it. When it comes to styles, you can directly manipulate inline styles using the style property, which lets you change specific CSS properties, such as element.style.color = 'blue' to change the text color. Additionally, you can set multiple styles at once using the cssText property. For managing classes, the classList property provides methods to add, remove, or toggle classes, making it easy to change the elementâ€™s appearance dynamically. Together, these techniques allow for flexible and interactive manipulation of web elements.
